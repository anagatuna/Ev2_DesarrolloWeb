Elementos Fundamentales de JavaScript

Tipos de datos

JavaScript tiene tipos de datos primitivos (string, number, boolean, null, undefined, symbol, bigint) y de referencia (object). 

let nombre = "Ana";   // string
                                    let edad = 30;        // number
                                    let activo = true;    // boolean
                                    let persona = { nombre: "Ana", edad: 30 }; // object
Variables

Se declaran con var, let o const. Usa let y const para un código más seguro. 

let contador = 1;
                                const PI = 3.1416;
Operadores

Permiten realizar operaciones matemáticas, lógicas y de comparación. 

let suma = 5 + 3;
                                let igual = (10 === "10"); // false
                                let logico = true && false; // false
Estructuras de control

Permiten tomar decisiones o repetir bloques de código: if, for, while. 

if (edad >= 18) {
                                  console.log("Mayor de edad");
                                } else {
                                  console.log("Menor de edad");
                                }
Funciones

Bloques reutilizables de código que pueden recibir parámetros y devolver valores. 

function saludar(nombre) {
                                      return "Hola " + nombre;
                                    }

                                    const despedir = (nombre) => "Adiós " + nombre;
Arrays

Colecciones ordenadas de valores. Pueden recorrer, filtrar o transformar datos fácilmente. 

const numeros = [1, 2, 3, 4];
                                    const dobles = numeros.map(n => n * 2);
Objetos

Agrupan propiedades y métodos en una sola estructura para representar entidades. 

const persona = {
                                  nombre: "Ana",
                                  edad: 25,
                                  saludar() { return "Hola " + this.nombre; }
                                };
DOM y eventos

Permiten manipular elementos HTML y escuchar eventos del usuario. 

document.querySelector("#btn")
                                .addEventListener("click", () => alert("Hola!"));
Asincronía y Promesas

Manejan procesos que tardan tiempo, como consultas a APIs, sin bloquear la ejecución. 

async function obtenerDatos() {
                                  const res = await fetch("/api");
                                  const data = await res.json();
                                  console.log(data);
                                }
Módulos

Dividen el código en archivos reutilizables mediante export e import. 

// math.js
                                export const suma = (a, b) => a + b;

                                // main.js
                                import { suma } from './math.js';
Manejo de errores

Usa try...catch para capturar errores y evitar que el programa se detenga. 

try {
                                      throw new Error("Algo salió mal");
                                    } catch (e) {
                                      console.error(e.message);
                                    }
