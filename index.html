Características

Lenguaje interpretado y del lado del cliente

El código se ejecuta directamente en el navegador del usuario, lo que agiliza la respuesta de las páginas web sin necesidad de recargar.

Multiparadigma

Es flexible y soporta varios estilos de programación, como la programación orientada a objetos, la programación imperativa y, especialmente, la programación funcional.

Tipado dinámico

No necesitas declarar el tipo de una variable (número, texto, etc.) al crearla. JavaScript lo determina automáticamente mientras se ejecuta el programa, lo que ofrece gran flexibilidad.

Basado en prototipos

En lugar de clases (como en otros lenguajes), la herencia de objetos se realiza a través de prototipos, lo que permite que los objetos hereden propiedades y métodos de otros objetos.

Asíncrono y "single-threaded"

A pesar de ejecutar tareas en un solo hilo, puede manejar operaciones asíncronas sin bloquear el resto del programa, gracias a su "Event Loop". Esto es crucial para mantener la fluidez de la interfaz de usuario.

Ecosistema enorme

Cuenta con una comunidad gigantesca y acceso a una inmensa cantidad de librerías (como React, Angular, Vue) y herramientas (como Node.js para el backend) que expanden sus capacidades a casi cualquier área del desarrollo de software.

Elementos Fundamentales

Variables Tipos de datos Operadores Estructuras de control Funciones Eventos y elementos DOM

Variables

Sirven para almacenar valores como números, texto, objetos, etc.

let nombre = "Chispi";   // variable que puede cambiar
const PI = 3.1416;       // variable constante (no cambia)
var edad = 25;           // es una forma antigua (evita usarla)

Tipos de Datos

Los principales tipos de datos en JavaScript son:

  • Number: representa números, tanto enteros como decimales: 42, 3.14
  • String: representa texto o cadenas de caracteres: "Hola, soy Chispi"
  • Boolean: representa valores lógicos verdadero o falso: true, false
  • Array: representa una lista o conjunto de elementos ordenados: [1, 2, 3]
  • Object: representa una estructura de datos con pares clave-valor: {nombre: "Chispi", edad: 9}

Operadores

Permiten realizar operaciones matemáticas y lógicas.

let a = 10;
let b = 5;

console.log(a + b); // 15 (suma)
console.log(a - b); // 5 (resta)
console.log(a * b); // 50 (multiplicación)
console.log(a / b); // 2 (división)
console.log(a % b); // 0 (módulo)
Operadores de comparación:
a > b      // true
a == 10    // true
a === "10" // false (por que tipo distinto)
a != b     // true

Estructuras de Control

Permiten tomar decisiones y repetir código.

Condicionales
let edad = 18;

if (edad >= 18) {
  console.log("Eres mayor de edad");
} else {
  console.log("Eres menor de edad");
}
Bucles
for (let i = 0; i < 3; i++) {
  console.log("Iteración " + i);
}
let n = 0;
while (n < 3) {
  console.log("n vale " + n);
  n++;
}

Funciones

Bloques de código que realizan una tarea específica.

function saludar(nombre) {
    return "Hola " + nombre;
}
console.log(saludar("Chispi"));

Eventos y Elementos del DOM

Permiten interactuar con los elementos HTML de la página.

Acceder a elementos:
// Por ID
const titulo = document.getElementById("titulo");

// Por clase
const notas = document.getElementsByClassName("nota");

// Por etiqueta
const parrafos = document.getElementsByTagName("p");

// Primer elemento que coincide con un selector
const primero = document.querySelector(".mensaje");

// Todos los que coinciden con un selector
const items = document.querySelectorAll(".item");
Ejemplo de evento:
<button id="boton">Haz clic</button>

<script>
document.getElementById("boton").addEventListener("click", () => {
  alert("¡Hiciste clic!");
});
</script>

Principales aplicaciones

JavaScript es uno de los lenguajes más utilizados en el desarrollo web. Permite crear sitios interactivos, dinámicos y con gran capacidad de respuesta, siendo esencial tanto en el frontend como en el backend.


Desarrollo Web

JavaScript da vida a las páginas web, permitiendo crear animaciones, menús dinámicos y formularios interactivos. Se usa junto con HTML y CSS en casi todos los sitios modernos.

Desarrollo del lado del servidor

Gracias a Node.js, JavaScript puede ejecutarse en el servidor para crear APIs, manejar bases de datos y procesar información en tiempo real.

Aplicaciones multiplataforma

Con frameworks como React Native, Ionic o Electron, JavaScript permite crear aplicaciones que funcionan tanto en dispositivos móviles como en computadoras, usando un mismo código base.

Inteligencia artificial y videojuegos

JavaScript también se usa para crear juegos en 2D y 3D con motores como Phaser o Three.js, e incluso para proyectos de inteligencia artificial con TensorFlow.js.
